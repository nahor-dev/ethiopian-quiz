🔹 About

Test your knowledge of Ethiopia with this interactive quiz app! 📝 Explore 10 categories including:

History 🏛️, Culture 🎭, Geography 🗺️, Language 🗣️, Food 🍲, Music 🎵, Sports ⚽, Politics 🏛️, Economy 💼, and Wildlife 🦁.

Built as a single-page application (SPA) with HTML, CSS, and JavaScript, fully responsive for mobile, tablet, and desktop, featuring dynamic questions, score tracking, and smooth navigation. 🎉

🌟 Features
✅ 10 Ethiopian-themed categories
✅ Dynamic multiple-choice quiz questions
✅ Score tracking and progress bar
✅ Fully responsive design
✅ Single-page navigation for smooth UX
✅ Easy to extend with new questions or categories

🚀 How to Use
<details> <summary>Instructions</summary>
Clone or download the repository:
git clone https://github.com/nahor-dev/ethiopian-quiz.git
Open index.html in your browser.
Select a category to start the quiz.
Answer questions and see your score.
Use Back buttons to navigate between screens or restart the quiz.
